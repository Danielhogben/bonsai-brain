# AI & Agent Infrastructure

- OpenClaw: primary agent gateway (systemd user service). Commands: `openclaw skills check`, `openclaw mcp serve`.
- Ollama: local LLM inference (often available at 192.168.0.245:11434 on this network).
- Unsloth Studio: GPU-accelerated training/inference on local GPU.

If automating agent operations, consult CLAUDE.md and GEMINI.md for environment-specific instructions and MCP tokens.
